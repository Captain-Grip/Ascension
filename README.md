# ASCENSION

Un Action-RPG en monde ouvert.

## Vision

Le joueur commence sans pouvoir.

Après avoir réussi les 10 Épreuves, il obtient son Permis d'Aventurier.

Il devra ensuite trouver un Maître afin de développer ses pouvoirs.
